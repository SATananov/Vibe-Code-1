          ❓ How to use

          • Type a Title, optionally add a note. Set ⭐ Important and ⏰ Urgent as needed.
          • Press Enter or click Add Task. Items appear in one of the four quadrants.
          • Keyboard: Tab to a task → Space/Enter toggles done, Delete removes, ← / → moves it.
          • Use the buttons: ✅ mark done, 🔁 move, 🗑 delete.
          • Everything is saved locally (localStorage). Works offline.
          • Toggle Theme (🌓) anytime.
